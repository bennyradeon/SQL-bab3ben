# Materi SQL Lanjutan (Interaktif) — DB Sekolah & DB Kuliah

## Cara pakai
1. Ekstrak file ZIP.
2. Buka `index.html` dengan browser (Chrome/Edge/Firefox).
3. Pilih tab **DB Sekolah** atau **DB Kuliah**.
4. Klik **Copy query** atau buka **Keterangan guru** untuk penjelasan + analogi.

## Struktur folder
- index.html
- assets/
  - styles.css
  - app.js
